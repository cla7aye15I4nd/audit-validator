# CertiK Audit Projects

1. For each new project, please **create a new project branch**, e.g. `<project-name>`. Make sure to avoid collision.
2. In the project branch, create a project folder under `projects/`, e.g. `projects/<example-project>`.
3. Put client source code under the created project folder, e.g. `projects/<example-project>/contracts`.

Note:
- The project branch doesn't need to be merged into `master`. Pull request isn't needed.
- Please don't include multiple projects in a single project branch.
- The `projects/example-project` folder is an example and can be removed.
