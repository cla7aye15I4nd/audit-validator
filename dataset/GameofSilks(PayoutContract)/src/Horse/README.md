# Silks-Contracts


# Horse 

- mintTransferReq
	- Function takes in ID's of YearlingPass Contract that want to be traded for Horses
	- Adds request based on id of msg.sender
- mintTransferFul
	- Function grabs request based on msg.sender
	- Uses data from request and random numbers generated by VRFConsumer to give out random Horses are preminted on contract and take in YearlingPass ID's

To Build & deploy code:
1. Run ```brownie compile``` off the root directory of the project
2. Run ```brownie run deploy.py --network goerli``` as an example replacing goerli with your network of choice

To run test:
1. After deployment run ```cd ./tests```
2. Run ```brownie test test_1.py --network goerli``` as an example replacing goerli with your network of choice
